## Helm 支持

- `nginx-ingress-controller`, `default-ingress` k8s ingress
- `metallb`, `slb`, `lb`
- `cronhpa`, `ali-cronhpa`, `kubernetes-cronhpa-controller`
- `tkn`， `tekton`
- `metrics-server`, `ms`, `kms` # 安装k8s默认安装
- `kubernetes_dashboard`, `kd` # 安装k8s默认安装
- `etcd`
